# MyChordpro

**MyChordpro** は、  
**ChordPro 形式のコード譜を、自分専用に保存・表示・練習するための私的 Web サイト**です。

このプロジェクトは、

- 自分で入力したコード譜だけを  
- 自分だけがログインして  
- 自分のブラウザで快適に表示・操作する  

ことを目的としています。

---

## 目的と位置づけ

MyChordpro は、**個人用のコード譜ビューア兼・練習支援ツール**です。

- ChordPro 形式データのレンダリング
- 移調、オートスクロール、表示調整
- YouTube 連携による再生補助
- 個人用メモ・手書き書き込み（ローカル保存）

を備えています。

### 明確に「やらないこと」

このサイトは、**オンラインでコード譜を共有するためのサービスではありません**。

- URL を他人に共有しても  
  **共有された人自身の MyChordpro 環境**が表示されます
- 他人の曲を見ることはできません
- 自分の登録した曲を公開・配布する機能はありません
- コミュニティ用途や公開利用は想定していません

**あくまで私的利用専用**のツールです。

---

## 認証とアクセス制御

- **GitHub 認証でログイン**します
- 未ログイン状態では、曲一覧・曲内容はいっさい表示されません
- ログイン後も、
  - **自分が登録したデータのみ**が対象になります
  - 他ユーザーの存在を意識する必要はありません

見た目はマルチユーザー対応ですが、  
**実態は完全に分離された個人用スペース**です。

---

## 想定している使い方

- 自分専用のコード譜置き場
- バンド練習・個人練習用の譜面表示
- 移調・オートスクロールを使った練習
- YouTube を横に置いた確認・参照
- 紙に書く代わりの個人メモ・書き込み

---

## 基本的な使い方

### トップページ

- 曲一覧の表示
- 曲名 / アーティスト名検索
- タグ検索（サジェストあり）
- ランキング表示

検索の補足：

- 通常入力：部分一致
- ダブルコーテーションで括ると `"完全一致"`：完全一致検索
- ドロップダウンを変更して、`target=tag`：タグ検索

---

### 曲ページ（譜面表示）

- ChordPro 形式のコード譜を譜面として表示
- 画面幅に応じて横スクロール対応
- 曲を開くと閲覧情報が内部的に更新されます

---

### Song Controls（練習補助）

- 移調・コード表記（♯ / ♭）
- オートスクロール（時間指定・速度調整）
- 表示カスタマイズ（コードサイズ・行間など）

---

### メモ・手書き

- 付箋メモの追加（タイトル / 本文 / 色）
- 譜面上への手書き書き込み
- Undo / 任意削除対応
- ピン留めによる譜面追従／画面固定

これらの情報はすべて：

- **このブラウザの localStorage のみに保存**
- サーバーや他端末と同期されません

---

### Tags / YouTube

- タグ一覧表示・タグ検索
- 登録済み YouTube 動画の再生
- 曲名 + アーティスト名での YouTube 検索

---

## 曲データの編集

ログインユーザーは、自分の曲を自由に管理できます。

- 新規追加 / 編集 / 削除
- タグ・YouTube の簡易編集

主な入力項目：

- `title`（曲名）
- `slug`（URL 用 ID）
- `artist`
- `tags`（1 行 1 タグ）
- `youtube`（1 行 1 動画）
- `chordPro`（譜面本文）

---

## データの考え方

- 曲データは **完全に自分専用**
- 他人と混ざらない
- URL を知っていても中身は見えない

MyChordpro は、

> 「自分の譜面を、オンラインに置いているだけ」

という感覚で使うためのものです。
